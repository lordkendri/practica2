1.	<!DOCTYPE>
Sirve para aclarar el tipo de documento con el que vamos a trabajar.

<!DOCTYPE html>
<html lang=”es”>
</html> 

2.	<head>
El código HTML insertado entre las etiquetas <html> se dividen en 2 secciones principales
el <head> y el <body>, el head es la cabecera donde usualmente se introducen elementos 
que no son visibles a los usuarios (back-end).

<!DOCTYPE html>
<html lang="es">
<head>
</head>
</html>

3.	<body>
Es la sección que representa la parte visible del Documento HTML o sea el cuerpo.

<!DOCTYPE html>
<html lang="es">
<head>
</head>
<body>
</body>
</html>



4.	<meta>
Va usualmente dentro de la cabecera <body> además tiene otras etiquetas como description,
keywords, que sirven para describir aspectos de la página web y con palabras claves facilitar
la localización de la web en los buscadores y el atributo name especifica el tipo y el atributo 
content su valor y ninguno de estos elementos se muestra en pantalla.

<!DOCTYPE html>
<html lang="es">
<head>

 <meta charset="iso-8859-1" />
 <meta name="description" content="Ejemplo de HTML5" />
 <meta name="keywords" content="HTML5, CSS3, JavaScript" />

</head>
<body>
</body>
</html>

5.	<base>
Especifica la dirección URL base que se utilizará para todas las direcciones URL relativas contenidas
dentro de un documento. (Sólo puede haber un elemento <base> en un documento).
…
<base href="http://www.example.com/page.html">
<base target="_blank" href="http://www.example.com/page.html">
…

7.	<link>
Otro importante elemento que va dentro de la cabecera del documento es <link>, este
elemento es usado para incorporar estilos, códigos Javascript, imágenes o iconos desde archivos
externos.

<!DOCTYPE html>
<html lang="es">
<head>

 <meta charset="iso-8859-1">
 <meta name="description" content="Ejemplo de HTML5">
 <meta name="keywords" content="HTML5, CSS3, JavaScript">

 <title>Este texto es el título del documento</title>

 <link rel=”stylesheet” href=”estilos.css”>

</head>
<body>
</body>
</html>

  <p>Las manzanas es un fruto que crece en un Manzano.</p>
